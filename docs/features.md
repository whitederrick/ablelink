## 직무지도원 기능

### 회원가입 / 인증

#### 셀프 가입 (전화번호 OTP)
- 전화번호 입력 → SMS OTP 발송 (5분 유효)
- OTP 확인 → 이름·비밀번호 입력 → 약관 동의 → 가입 완료
- 가입 후 자동 로그인 → `/worker/site/register` 리다이렉트

#### 초대 링크 가입
- 관리자가 전화번호로 초대 링크 생성 (24시간 유효, 6자리 코드)
- 직무지도원: 링크 접속 → 코드 입력 → 정보 입력 → 약관 동의 → 가입
- siteId 지정된 경우 SiteAssignment 자동 생성 → `/worker/home` 리다이렉트

#### 계약서 서명 후 가입 (레거시)
- 관리자가 카카오 알림톡으로 계약서 서명 링크 발송
- 서명 완료 시 임시 비밀번호 발급 → 온보딩 플로우 진행

#### 임시 계정 온보딩
- `isTemporary=true` 상태로 로그인 시 → `/worker/onboarding` 강제
- 이름·전화번호 확인, 비밀번호 변경 → `isTemporary=false` 전환

---

### GPS 출퇴근

- 출근 버튼 클릭 → 현재 GPS 위치 수집
- 현장 기준점 반경 내/외 판정 (기본 100m)
- DailyAttendance 생성 (startTime, 위도/경도 기록)
- 퇴근 시 endTime 기록 + 훈련생별 일지 연결
- 범위 외 출퇴근 → AttendanceIssue 자동 생성
- `AUTO_FINALIZE_MINUTES`(기본 60분) 경과 후 자동 확정

---

### 업무일지

#### 수동 작성 (FREE+)
- 훈련생별 1:1 지도 시간, 그룹 지도 시간 입력
- 수행 과업명·달성도 점수 기록
- 임시저장 / 완료 상태 구분

#### AI 음성 일지 (STARTER+)
- 음성 녹음 → Groq Whisper STT → Gemini 일지 자동 생성
- 문장 수 선택 (간략/보통/상세)
- 일괄 AI 작성: 미작성 일지 전체에 동일 음성 적용

### 문서 조회 (5종)
1. **출근부** — 월별 출퇴근 시간표

3. **훈련생 종합평가** — 훈련생별 평가 점수
4. **적응지도 일지** — ADAPTATION 단계 일지
5. **적응지도 종합평가** — 적응지도 평가 점수

## 관리자 기능

### 현장 관리
- 현장 등록 (주소 → GPS 자동 변환)
- GPS 기준점 관리 (관리자 설정 / 직무지도원 제안 → 승인)
- 현장 담당자 연락처 관리

### 직무지도원 관리
- 목록 조회·검색 (이름·전화번호·현장명)
- 이름·전화번호 수정, 임시 비밀번호 발급
- 초대 링크 생성·발송 (전화번호, 선택: 이름·현장 지정)

### 배정 관리
- 직무지도원 ↔ 현장 배정 등록·수정
- 근무형태 설정 (AM/PM/FULL_DAY/CUSTOM)
- 출퇴근 지도 포함 여부

### 출퇴근 관리
- 일별/직무지도원별 출퇴근 기록 조회
- 지도 시각화 (GPS 위치 확인)
- 근태 이슈 수신함: 범위 외·시간 이상 이슈 확인·소명 요청·해결

### 근로계약서 (STARTER+)
- 계약서 생성 (계약 기간·근무형태 입력)
- 카카오 알림톡 발송 (서명 링크)
- 직무지도원 서명 완료 후 관리자 서명 → PDF 생성

### 문서 관리
- DocumentRun 생성 (제출 기한 설정)
- PDF 미리보기 / 생성 / 관리자 서명
- 이메일 발송 (AWS SES)
- 발송 이력 조회

### 급여 관리 (STANDARD+)
- 급여 계약 등록 (내부/외부, 시급/일급/월급, 사업소득/근로소득)
- 훈련생 2명+ 시 별도 시급 적용
- 월별 급여 계산 실행 → DRAFT 생성 → 검토 후 확정
- 에이전시 커스텀 공제 항목 CRUD
- 4대보험 요율 연도별 관리

### 훈련생 진척도 리포트 (STANDARD+)
- 현장별·훈련생별 일지 작성 현황
- 평가 점수 추이

### 감사 서류 패키지 (STANDARD+)
- 직무지도원별 특정 기간 전체 문서를 ZIP으로 다운로드
- 포함: 출근부, 훈련일지, 종합평가, 적응지도 일지, 적응지도 평가

### CSV 내보내기 (FREE+)
- 출퇴근 기록 Raw CSV 다운로드

### 구독 관리
- 플랜 업그레이드/다운그레이드
- 토스페이먼츠 빌링 연동
- 구독 해지

|------|------|------|
